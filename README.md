# 🗂️ SmartApp: Помощник по складыванию одежды для Сбера

Приложение, которое поможет вам научиться складывать вещи красиво, как в магазине! Интегрировано с голосовым ассистентом Сбера.

## ✨ Функции

### Основные возможности:
1. **Инструкции по складыванию** - для каждой категории вещей есть заготовленные инструкции
2. **Озвучивание советов** - приложение зачитывает вслух инструкции по складыванию и стирке
3. **Советы по стирке** - автоматические рекомендации по уходу за одеждой
4. **Напоминания об уходе** - можно установить дату следующего ухода за вещью
5. **Голосовое управление** - полная интеграция с ассистентом Сбера

### Дополнительные функции:
- **Категории вещей**: Верх, Низ, Нижнее бельё, Носки, Шерсть, Другое
- **Пользовательские инструкции** - возможность добавить свои советы
- **Статус готовности** - отмечайте сложенные вещи
- **Визуальные уведомления** - анимация для вещей, требующих ухода

## 🤖 Интеграция со Сбером

### Получение токена из SaluteDev

1. Откройте [SaluteDev](https://smartapp-code.sberdevices.ru/)
2. Авторизуйтесь через Sber ID
3. Создайте новый проект (SmartApp)
4. Перейдите в раздел **"Настройки"** → **"Безопасность"** или **"Токены"**
5. Нажмите **"Сгенерировать токен"** или скопируйте существующий
6. Сохраните токен — он понадобится для файла `.env`

> ⚠️ **Важно**: Токен является секретным ключом. Не публикуйте его в репозитории!

### Настройка проекта в SaluteDev

1. В созданном проекте укажите:
   - **Имя SmartApp**: например, `wardrobe`
   - **Entry Point**: URL вашего развёрнутого приложения (например, `https://your-app.vercel.app`)
   - **Конфигурация**: загрузите файл `sberbot.json` или настройте intents вручную
2. Сохраните настройки

### Переменные окружения

Скопируйте файл `.env.example` в `.env` и заполните своими значениями:

```bash
cp .env.example .env
```

Затем отредактируйте `.env`:

```env
REACT_APP_TOKEN="ваш_токен_от_SaluteDev"
REACT_APP_SMARTAPP="имя_вашего_smartapp"
```

## 📱 Как использовать

### Через веб-интерфейс:
1. **Добавить вещь**: Введите название, выберите категорию
2. **Получить инструкцию**: Нажмите "🔊 Складка" чтобы услышать как сложить
3. **Узнать про стирку**: Нажмите "🔊 Стирка" для совета по уходу
4. **Отметить готово**: Нажмите "✅ Готово" когда сложили вещь
5. **Установить напоминание**: Выберите дату следующего ухода

### Через голосового ассистента:
1. Скажите: "Запусти wardrobe" (или имя вашего smartapp)
2. Произнесите команду, например: "Добавить футболку"
3. Для управления используйте голосовые команды из списка выше

## 📋 Конфигурация SmartApp

Файл `sberbot.json` содержит полную конфигурацию для Сбера:

- **intents**: Описание распознаваемых намерений
- **slots**: Параметры для каждого намерения
- **commands**: Маппинг команд на действия приложения
- **greetings/goodbyes**: Приветствия и прощания

## 🔐 Безопасность

- Токен для интеграции хранится в `.env` файле
- Данные пользователя сохраняются в localStorage браузера
- Рекомендуется использовать HTTPS для продакшена
