# 📖 Руководство по развертыванию SmartApp для Сбера

## Шаг 1: Подготовка проекта

### 1.1 Установка зависимостей
```bash
npm install --legacy-peer-deps
```

### 1.2 Настройка переменных окружения
Создайте файл `.env` в корне проекта:
```env
REACT_APP_TOKEN="ваш_токен_от_SaluteDev"
REACT_APP_SMARTAPP="имя_вашего_smartapp"
```

## Шаг 2: Регистрация в SaluteDev

### 2.1 Создание проекта
1. Перейдите на [SaluteDev](https://smartapp-code.sberdevices.ru/)
2. Войдите через Sber ID
3. Нажмите "Создать новый проект"
4. Выберите тип проекта: "Web App"

### 2.2 Настройка проекта
1. **Название**: Помощник по складыванию одежды
2. **Описание**: SmartApp поможет вам научиться складывать вещи красиво
3. **Entry Point**: URL вашего размещённого приложения (например, https://your-domain.com)

### 2.3 Импорт конфигурации
1. Откройте файл `sberbot.json` из этого проекта
2. Скопируйте содержимое раздела `intents`
3. Вставьте в разделе "Intents" в SaluteDev
4. Повторите для `commands`, `greetings`, `goodbyes`

## Шаг 3: Локальная разработка

### 3.1 Запуск SmartApp Debugger
```bash
npm start
```

Приложение откроется в браузере с панелью отладки SaluteJS.

### 3.2 Тестирование голосовых команд
В панели отладки:
1. Введите тестовую фразу в поле ввода
2. Нажмите Enter или кнопку отправки
3. Проверьте реакцию приложения

Примеры тестовых фраз:
- "Запусти wardrobe"
- "Добавить футболку"
- "Готово"
- "Как сложить"

## Шаг 4: Сборка для продакшена

### 4.1 Создание production сборки
```bash
npm run build
```

### 4.2 Размещение на хостинге
Скопируйте содержимое папки `build` на ваш хостинг:

**Варианты хостинга:**
- GitHub Pages
- Netlify
- Vercel
- Собственный сервер с HTTPS

### 4.3 Настройка HTTPS
Для работы с ассистентом Сбера требуется HTTPS:
- Используйте Let's Encrypt для бесплатного SSL
- Или разместите на платформе с автоматическим HTTPS (Netlify, Vercel)

## Шаг 5: Публикация в каталоге Сбера

### 5.1 Подготовка к публикации
1. Пройдите модерацию в SaluteDev
2. Подготовьте скриншоты приложения
3. Напишите подробное описание

### 5.2 Подача заявки
1. В личном кабинете разработчика Сбера
2. Заполните форму публикации
3. Прикрепите необходимые документы
4. Ожидайте модерацию (обычно 3-5 рабочих дней)

## 🔧 Решение проблем

### Ошибка: "Failed to initialize assistant"
- Проверьте токен в `.env`
- Убедитесь, что имя smartapp совпадает
- Проверьте консоль браузера на ошибки

### Ошибка CORS
- Убедитесь, что сервер отдаёт правильные заголовки
- Для локальной разработки используйте прокси

### Проблемы с голосовыми командами
- Проверьте настройку intents в SaluteDev
- Убедитесь, что слова не находятся в `ASSISTANT_IGNORED_WORDS`
